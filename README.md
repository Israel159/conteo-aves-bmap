# 🐦 Detector Automático de Aves - BMAP APECO

Sistema de inteligencia artificial para detectar 16 especies de aves marinas y costeras peruanas.

## Especies detectables

- chuita, chuita adulta
- cushuri adulto, cushuri juvenil
- gallinazo cabeza roja
- gaviota peruana adulta
- guanay adulto
- pelicano adulto, pelicano juvenil
- pichon pinguino, pichon piquero
- pinguino adulto, pinguino juvenil
- piquero adulto, piquero juvenil
- zarcillo

## Uso

1. Sube una o más imágenes JPG/PNG
2. Ajusta los umbrales de confianza e IoU si es necesario
3. Revisa las detecciones y descarga los resultados en CSV

## Modelo

YOLOv8n (26n.pt) entrenado por el equipo BMAP APECO.
